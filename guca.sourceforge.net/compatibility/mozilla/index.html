Version Platform Compatibility Notes
1.0 Win 98, Win 98 SE, Win ME, Win NT4 Poor Although support for Unicode is included, there is no support for the complex rendering required by Gurmukhi.
1.0 Win 2000 Okay Provided you have upgraded Uniscribe you should be able to view Gurmukhi text as you can on Windows XP. If you have not upgraded Uniscribe, complex rendering required for Gurmukhi support is not possible.
1.0 Win XP Good Overall support for Gurmukhi is good. However there are problems with the use of ZWJ and ZWNJ, and major problems occur when using justified text.
1.0 --enable-pango Linux Good Supports Gurmukhi if compiled with the '--enable-pango' switch. However there are problems with the use of ZWJ and ZWNJ, and major problems occur when using justified text. The standard version does not support Gurmukhi.
